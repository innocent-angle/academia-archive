Equipment Guide

 

Here we will compile a list of some commonly used equipment, and their budget.

Please note that prices vary based on quality. Expect to pay more if you want higher quality. 

 

Make sure to buy ammo!

 

Combat Set + Enfield + Diamond Bladed Knife (free in the armory)

Note: We have these available in the armory, for free!

This is a solid set that myself and many others used for years.
Don't feel pressured to upgrade from this, especially if you are new.

 

  • Enfield SA-80*
  • Diamond Bladed Knife
  • Combat Helmet*
  • Combat Vest
  • Combat Pants
  • Combat Gloves
  • Combat Boots

 

*ArmaLite M15-A4 is a good upgrade from the Enfield

*Motorcycle helmets are a good upgrade from the combat helmet

 

Dune / Riot Sets ( ~420 million)

Personally I don't find these sets worth it. Some people combine these with assault pieces.

Combining pieces from different sets causes you to lose the full set bonus. 

 

  • ArmaLite M15-A4*
  • Dune Helmet
  • Dune Vest
  • Dune Pants
  • Dune Gloves
  • Dune Boots

 

*Consider a RW weapon instead of an ArmaLite

 

Assault Set (~$1 Billion)

 

  • RW Weapon
  • Assault Helmet
  • Assault Body
  • Assault Pants
  • Assault Gloves
  • Assault Boots

 

Delta, Vanguard, Marauder, EOD (Tens of billions)

These are all for the ultra rich, not much to say here.
